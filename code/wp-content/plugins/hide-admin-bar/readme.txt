== Installation ==
1. Download the hide-admin-bar.zip file to your computer.
1. Unzip the file.
1. Upload the hide-admin-bar folder to your `/wp-content/plugins/` directory.
1. Activate the plugin through the *Plugins* widget in WordPress.
